Sistemas de control PID

Un sistema de control busca como su mismo nombre lo dice controlar el estado o la salida de un sistema

Definicion en wikipedia

En la vida cotidiana podemos encontrar diferentes tipos de sistemas,
desde sistemas biologicos, hasta sistemas electricos o mecanicos. Un sistema puede ser desde un simple circuito RC en electronica, hasta un sistema mecanico no lineal controlado por un motor electrico




Proporcional Efectua la accion de control con base en el error
Integral Efectua la accion de control con base en la integral del error
Derivativo Efectua la accion de control con base la derivada del error
